# Eisenhower Matrix for Obsidian

A simple but powerful **Eisenhower Matrix** plugin for task management inside Obsidian.

## âœ¨ Features
- Four quadrants: **Do First, Schedule, Delegate, Eliminate**
- Add, complete, and delete tasks directly in the matrix
- **Drag & drop** tasks between quadrants or reorder inside the same quadrant
- Obsidian **wiki-links `[[Note]]`** and **Markdown links `[link](url)`** are clickable
- **Task counters** in each quadrant
- Toggle **Hide Completed Tasks**
- **Quick search** across tasks
- Customizable color scheme (can be toggled in settings)
- Infinite tasks supported, with scrollable lists
- All tasks are stored in a single markdown file (default: `EisenhowerTasks.md`)
